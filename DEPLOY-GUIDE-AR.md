# دليل نشر منصة ندى موسى — خطوة بخطوة 🚀

## ما بداخل الحزمة
```
├── netlify.toml                     ← إعدادات Netlify (لا تعدليه)
├── public/
│   ├── index.html                   ← صفحة أعمالك (الرئيسية)
│   └── checker.html                 ← فاحص السيرة الذاتية و LinkedIn
└── netlify/functions/
    └── analyze.mjs                  ← الوسيط الآمن لمفتاح API
```

## الخطوة 1: مفتاح API (مرة واحدة — 5 دقائق)
1. ادخلي إلى **console.anthropic.com** وأنشئي حساباً (منفصل عن اشتراك Claude.ai)
2. من **Billing** أضيفي رصيداً — ابدئي بـ **$5 فقط** (تكفي لمئات الفحوص التجريبية)
3. من **API Keys** اضغطي **Create Key** وانسخي المفتاح (يبدأ بـ `sk-ant-`)
   ⚠️ **لا تضعي المفتاح في أي ملف أبداً** — فقط في إعدادات Netlify (الخطوة 3)

## الخطوة 2: رفع الموقع
**الطريقة السهلة (سحب وإفلات):**
1. ادخلي **app.netlify.com** → **Add new site → Deploy manually**
2. اسحبي **مجلد المشروع كاملاً** (وليس مجلد public وحده!)

**الطريقة الأفضل (GitHub — تحديثات تلقائية):** ارفعي المجلد لمستودع GitHub واربطيه من
**Add new site → Import an existing project**

> 💡 درس نتلفاي الذي تعرفينه من منصة SHRM: إن رفعتِ يدوياً وظهرت صفحة فارغة، فالسبب
> Publish directory — في حالتنا `netlify.toml` يضبطها تلقائياً على `public` فلا تغيّريها.

## الخطوة 3: ربط المفتاح (الأهم!)
1. من لوحة موقعك: **Site configuration → Environment variables → Add a variable**
2. الاسم: `ANTHROPIC_API_KEY` (بهذا الشكل حرفياً)
3. القيمة: مفتاحك `sk-ant-...`
4. احفظي ثم **Deploys → Trigger deploy → Deploy site** (إعادة نشر لتفعيل المتغير)

## الخطوة 4: التجربة
- افتحي `اسم-موقعك.netlify.app` → صفحة أعمالك
- اضغطي "افحصي سيرتك مجاناً" → جربي فحصاً كاملاً
- إن ظهر خطأ "ANTHROPIC_API_KEY غير مضبوط" → راجعي الخطوة 3

## الخطوة 5 (اختياري): دومين باسمك
**Domain management → Add a domain** — شراء nadamusa.com (~$12/سنة) يرفع مصداقية العلامة كثيراً.

## التكاليف المتوقعة
| البند | التكلفة |
|---|---|
| استضافة Netlify | مجانية (تكفي حتى ~100 ألف زيارة) |
| الفحص الواحد (نداءان) | ~2-4 سنتات |
| 100 فحص شهرياً | ~$3-4 |
| 1000 فحص شهرياً | ~$30-40 |

**قاعدة أمان:** من console.anthropic.com → **Limits** ضعي حداً شهرياً للإنفاق
(مثلاً $25) حتى لا تتفاجئي أبداً بفاتورة.

## ملاحظات تقنية
- **الحسابات والأرصدة**: على الموقع تُحفظ في متصفح المستخدمة (localStorage) — مناسبة
  للإطلاق التجريبي. الحسابات الحقيقية المؤمّنة (تحقق بالبريد + قاعدة بيانات) نبنيها
  عند إضافة الاشتراكات المدفوعة.
- **المفتاح آمن**: كل النداءات تمر عبر `/api/analyze` في الخادم — المتصفح لا يرى المفتاح إطلاقاً.
- نفس ملف `checker.html` يعمل داخل Claude وعلى Netlify — يكتشف بيئته تلقائياً.

## عند أي مشكلة
انسخي لي نص الخطأ حرفياً + لقطة شاشة من **Deploys → (آخر نشرة) → Deploy log** وسأشخّصها فوراً.
